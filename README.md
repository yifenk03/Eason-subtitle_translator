
# LM Studio 字幕翻译工具

使用本地大语言模型（LM Studio等）对字幕文件进行智能翻译的 GUI 工具。

## 功能特点

- **支持多种格式**：TXT、SRT、ASS 字幕格式
- **本地 LLM**：调用本地 LM Studio 模型，保护隐私
- **批量翻译**：支持单个文件和文件夹批量处理
- **串行处理**：确保模型同时只有一个调用
- **GPU 监控**：实时显示显卡显存使用情况
- **详细日志**：实时显示翻译进度和状态

## 安装

### 1. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 2. 安装并启动 LM Studio

1. 下载 [LM Studio](https://lmstudio.ai/)
2. 下载所需的翻译模型（如日语模型）
3. 启动 LM Studio 并加载模型
4. 确保 API 服务运行在 `http://127.0.0.1:1234/v1`

## 使用方法

### 启动程序

```bash
python subtitle_translator.py
```

### 基本操作

1. **连接 LM Studio**
   - 点击「检测连接」按钮
   - 等待状态显示「已连接」

2. **选择文件**
   - 点击「选择字幕文件」选择单个文件
   - 或点击「选择文件夹」批量处理

3. **设置保存格式**
   - 选择输出格式（SRT 或 ASS）

4. **开始翻译**
   - 点击「开始翻译」按钮
   - 翻译完成后文件保存在原文件同目录

## 提示词说明

默认提示词设置了以下翻译规则：

| 规则 | 说明 |
|------|------|
| 保留格式 | 时间戳和序号保持不变 |
| 标点处理 | 句末只保留问号，其他标点去除 |
| 上下文连贯 | 结合前后字幕确保通顺 |
| 文风 | 口语化、自然的中文表达 |

可点击「恢复默认模板」重置。

## 常见问题

### Q: 检测连接失败？

1. 确认 LM Studio 已启动
2. 确认已在 LM Studio 中加载模型
3. 检查 API 地址是否正确（默认 `http://127.0.0.1:1234/v1`）

### Q: 如何增加 Max Tokens？

在设置中修改 Max Tokens 值，建议 4096 以上。

### Q: 显存不足？

1. 在 LM Studio 中使用量化模型
2. 降低同时处理的字幕数量
